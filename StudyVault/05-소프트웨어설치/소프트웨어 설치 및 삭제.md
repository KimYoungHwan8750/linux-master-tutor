## RPM (RedHat Package Manager)

| 옵션 | 기능 |
|------|------|
| `-i` | 설치 (install) |
| `-U` | 업그레이드 (없으면 설치) |
| `-e` | 삭제 (erase) |
| `-q` | 질의 (query) |
| `-qa` | 설치된 전체 패키지 목록 |
| `-qi` | 패키지 상세 정보 |
| `-ql` | 패키지 파일 목록 |
| `-V` | 검증 (verify) |
| `-v` | 상세 출력 |
| `-h` | 진행률 표시 (#) |

```bash
rpm -ivh package.rpm      # 설치 (상세+진행률)
rpm -Uvh package.rpm      # 업그레이드
rpm -e package            # 삭제
rpm -qa | grep httpd      # httpd 관련 패키지 검색
```

> [!warning] RPM의 한계
> 의존성 자동 해결이 안 됨 → 의존 패키지를 수동으로 먼저 설치해야 함

## YUM (Yellowdog Updater Modified)

| 명령어 | 기능 |
|--------|------|
| `yum install package` | 설치 |
| `yum update package` | 업데이트 |
| `yum remove package` | 삭제 |
| `yum search keyword` | 검색 |
| `yum list installed` | 설치된 목록 |
| `yum info package` | 패키지 정보 |

- **의존성 자동 해결** — RPM과 가장 큰 차이
- 설정 파일: `/etc/yum.repos.d/`

## DPKG (Debian Package)

| 옵션 | 기능 |
|------|------|
| `-i` | 설치 |
| `-r` | 삭제 (설정 유지) |
| `-P` | 완전 삭제 (purge) |
| `-l` | 설치된 패키지 목록 |
| `-L` | 패키지 파일 목록 |

## APT-GET

| 명령어 | 기능 |
|--------|------|
| `apt-get install package` | 설치 |
| `apt-get update` | 패키지 목록 갱신 |
| `apt-get upgrade` | 전체 업그레이드 |
| `apt-get remove package` | 삭제 |
| `apt-get purge package` | 완전 삭제 |

## 소스 코드 설치

```
tar xvf source.tar.gz     # 1. 압축 해제
cd source/                 # 2. 디렉토리 이동
./configure               # 3. 환경 설정 (Makefile 생성)
make                      # 4. 컴파일
make install              # 5. 설치
```

> [!important] 순서가 핵심
> `./configure → make → make install` 순서 반드시 지켜야 함!

## 압축/아카이브

### tar (아카이브)
| 옵션 | 기능 |
|------|------|
| `-c` | 생성 (create) |
| `-x` | 해제 (extract) |
| `-v` | 상세 출력 |
| `-f` | 파일명 지정 |
| `-z` | gzip 압축/해제 |
| `-j` | bzip2 압축/해제 |
| `-J` | xz 압축/해제 |

### 압축 도구 비교
| 도구 | 확장자 | 압축률 | 속도 |
|------|--------|--------|------|
| gzip | .gz | 보통 | 빠름 |
| bzip2 | .bz2 | 높음 | 보통 |
| xz | .xz | 최고 | 느림 |

```bash
tar czvf archive.tar.gz dir/    # gzip 압축
tar cjvf archive.tar.bz2 dir/   # bzip2 압축
tar cJvf archive.tar.xz dir/    # xz 압축
```
