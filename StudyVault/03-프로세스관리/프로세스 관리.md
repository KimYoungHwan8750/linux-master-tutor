# 프로세스 관리 (★★★)

#process #linux-master

## Overview Table (한눈에 비교)
| 구분 | 포그라운드 | 백그라운드 |
|------|-----------|-----------|
| 실행 | 터미널 점유 | `&` 붙여 실행 |

| 전환 | Ctrl+Z → `bg` | `fg`로 전환 |

## 프로세스 확인 명령어

| 명령어 | 용도 | 주요 옵션 |
|--------|------|-----------|
| `ps` | 프로세스 목록 | `aux` (전체), `-ef` (전체, 풀포맷) |
| `pstree` | 트리 형태 출력 | `-p` (PID 표시) |
| `top` | 실시간 모니터링 | |
| `jobs` | 백그라운드 작업 목록 | |

## 프로세스 제어

| 명령어 | 기능 |
|--------|------|
| `command &` | 백그라운드 실행 |
| `fg %N` | N번 작업을 포그라운드로 |
| `bg %N` | N번 작업을 백그라운드로 |
| `nohup command &` | 로그아웃 후에도 계속 실행 |

## kill / 시그널

| 시그널 | 번호 | 의미 |
|--------|------|------|
| SIGHUP | 1 | 터미널 끊김 / 재시작 |
| SIGINT | 2 | 인터럽트 (Ctrl+C) |
| SIGKILL | 9 | **강제 종료** (무시 불가) |
| SIGTERM | 15 | **정상 종료** (기본값) |
| SIGSTOP | 19 | 일시 정지 |

```
kill -9 PID        # 강제 종료
kill -15 PID       # 정상 종료 (기본)
kill PID           # -15와 동일
killall httpd      # 이름으로 종료
```

> [!warning] kill의 기본 시그널
> `kill PID`는 `-15`(SIGTERM)이 기본. `-9`(SIGKILL)는 명시해야 함.

## nice / renice

프로세스 **우선순위(NI값)** 조정.

- 범위: **-20**(최고 우선순위) ~ **19**(최저 우선순위)
- 기본값: 0

| 명령어 | 용도 | 제한 |
|--------|------|------|
| `nice -n 10 command` | 우선순위 10으로 실행 | 일반 사용자: 0~19만 |
| `renice 5 -p PID` | 실행 중 프로세스 변경 | root만 우선순위 높일 수 있음 |

> [!important] NI 값 혼동 주의
> NI 값이 **낮을수록** 우선순위가 **높음** (직관과 반대!)

## 데몬(Daemon) 실행 방식

| 방식 | 특징 |
|------|------|
| **standalone** | 항상 메모리에 상주, 빠른 응답, 리소스 많이 사용 |
| **inetd (xinetd)** | 요청 시에만 실행, 리소스 절약, 응답 느림 |

> [!tip] standalone vs inetd 선택 기준
> 자주 사용하는 서비스(httpd, sshd) → standalone / 가끔 사용하는 서비스 → inetd

## crontab (작업 스케줄링)

```
분(0-59)  시(0-23)  일(1-31)  월(1-12)  요일(0-6)  명령
```

| 특수문자 | 의미 |
|----------|------|
| `*` | 매번 |
| `,` | 여러 값 (1,3,5) |
| `-` | 범위 (1-5) |
| `/` | 간격 (*/5 = 5분마다) |

```
# 매일 새벽 3시에 백업
0 3 * * * /home/backup.sh

# 월~금 9시에 실행
0 9 * * 1-5 /home/work.sh
```

| 명령어 | 용도 |
|--------|------|
| `crontab -e` | 크론탭 편집 |
| `crontab -l` | 크론탭 목록 |
| `crontab -r` | 크론탭 삭제 |

## Exam/Test Patterns (시험 빈출 패턴)
| Scenario/Keyword | Answer |
|-------------------|--------|
| "프로세스 강제 종료" | **kill -9 PID** |
| "kill 기본 시그널" | **15 (SIGTERM)** |
| "로그아웃 후 계속 실행" | **nohup** |
| "우선순위 높이기(NI 낮추기)" | **root만 가능** |
| "항상 메모리 상주 데몬" | **standalone** |
| "매일 새벽 2시 실행 cron" | `0 2 * * * command` |
| "프로세스 트리 출력" | **pstree** |
