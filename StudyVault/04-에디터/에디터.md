## vi/vim 에디터

### 3가지 모드

```
                    i, a, o, I, A, O
  ┌──────────┐  ──────────────────→  ┌──────────┐
  │ 명령 모드  │                       │ 입력 모드  │
  │ (Command) │  ←──────────────────  │ (Insert) │
  └──────────┘        Esc            └──────────┘
       ↕
    : (콜론)
       ↕
  ┌──────────┐
  │ EX 모드   │
  │ (라인 모드)│
  └──────────┘
```

| 모드 | 설명 | 진입 방법 |
|------|------|----------|
| **명령 모드** (Command) | 커서 이동, 복사, 삭제, 검색 등 | vi 시작 시 기본 모드, Esc로 복귀 |
| **입력 모드** (Insert) | 텍스트 입력/편집 | i, a, o, I, A, O |
| **EX 모드** (라인 모드) | 저장, 종료, 치환, 설정 등 | :(콜론) 입력 |

### 입력 모드 전환 키 ★★

| 키 | 설명 | 커서 위치 |
|----|------|----------|
| **i** | 커서 **앞**에서 입력 (insert) | 현재 위치 앞 |
| **I** | 줄 **시작** 위치에서 입력 | 줄 맨 앞 |
| **a** | 커서 **뒤**에서 입력 (append) | 현재 위치 뒤 |
| **A** | 줄 **끝** 위치에서 입력 | 줄 맨 뒤 |
| **o** | 현재 줄 **아래**에 새 줄 삽입 후 입력 | 아래 새 줄 |
| **O** | 현재 줄 **위**에 새 줄 삽입 후 입력 | 위 새 줄 |

> [!tip] 외우기 팁
> - **i** (insert) = 앞, **a** (append) = 뒤
> - 소문자 = 커서 기준, **대문자 = 줄 기준**
> - **o** = 아래(open below), **O** = 위(open above)

### 명령 모드 - 삭제

| 키 | 설명 |
|----|------|
| **x** | 커서 위치의 **문자 1개** 삭제 |
| X | 커서 앞 문자 1개 삭제 |
| **dd** | 현재 **줄 전체** 삭제 (잘라내기) |
| ndd | n줄 삭제 (예: 5dd = 5줄 삭제) |
| **dw** | 커서부터 **단어 끝**까지 삭제 |
| d$ | 커서부터 줄 끝까지 삭제 |
| d0 | 커서부터 줄 처음까지 삭제 |
| dG | 커서부터 파일 끝까지 삭제 |

### 명령 모드 - 복사/붙여넣기

| 키 | 설명 |
|----|------|
| **yy** | 현재 **줄 복사** (yank) |
| nyy | n줄 복사 (예: 3yy = 3줄 복사) |
| yw | 커서부터 단어 끝까지 복사 |
| **p** | 커서 **아래**(뒤)에 붙여넣기 (paste) |
| **P** | 커서 **위**(앞)에 붙여넣기 |

### 명령 모드 - 기타

| 키 | 설명 |
|----|------|
| u | 실행 취소 (undo) |
| Ctrl+r | 다시 실행 (redo, vim만) |
| . | 마지막 명령 반복 |
| J | 현재 줄과 다음 줄 합치기 (join) |
| r | 커서 위치 문자 1개 교체 (replace) |
| R | 입력하는 문자로 계속 교체 (replace mode) |
| ~ | 대소문자 전환 |

### 커서 이동

| 키 | 설명 |
|----|------|
| **h** | ← 왼쪽 |
| **j** | ↓ 아래 |
| **k** | ↑ 위 |
| **l** | → 오른쪽 |
| w | 다음 단어 처음으로 |
| b | 이전 단어 처음으로 |
| e | 현재 단어 끝으로 |
| 0 | 줄 처음으로 |
| $ | 줄 끝으로 |
| **G** | 파일 **마지막 줄**로 |
| **gg** | 파일 **첫 줄**로 (1G와 동일) |
| nG | n번째 줄로 |
| Ctrl+f | 한 화면 앞으로 (page down) |
| Ctrl+b | 한 화면 뒤로 (page up) |
| H | 화면 맨 위로 (High) |
| M | 화면 중간으로 (Middle) |
| L | 화면 맨 아래로 (Low) |

### 검색

| 키 | 설명 |
|----|------|
| **/pattern** | 아래 방향으로 pattern 검색 |
| ?pattern | 위 방향으로 pattern 검색 |
| **n** | 같은 방향으로 다음 검색 결과 |
| **N** | 반대 방향으로 다음 검색 결과 |

### EX 모드 - 치환 ★★

```vim
:s/old/new/          " 현재 줄의 첫 번째 old를 new로 치환
:s/old/new/g         " 현재 줄의 모든 old를 new로 치환
:%s/old/new/g        " 파일 전체에서 모든 old를 new로 치환 ★
:%s/old/new/gc       " 파일 전체, 확인 후 치환 (confirm)
:n,ms/old/new/g      " n줄부터 m줄까지 치환
```

| 구문 | 범위 |
|------|------|
| :s | 현재 줄만 |
| :%s | **파일 전체** |
| :n,ms | n줄 ~ m줄 |
| g | 줄 내 **모든** 매칭 (없으면 첫 번째만) |
| c | 확인(confirm) 후 치환 |

### EX 모드 - 저장/종료

| 명령 | 설명 |
|------|------|
| **:w** | 저장 (write) |
| :w 파일명 | 다른 이름으로 저장 |
| **:q** | 종료 (quit) |
| **:q!** | 저장하지 않고 **강제 종료** |
| **:wq** | 저장 후 종료 |
| :wq! | 강제 저장 후 종료 |
| **:x** | 저장 후 종료 (:wq와 동일) |
| ZZ | 저장 후 종료 (명령 모드에서, :없이) |

### EX 모드 - 환경 설정

| 명령 | 설명 |
|------|------|
| **:set nu** | 줄 번호 표시 (number) |
| **:set nonu** | 줄 번호 숨김 |
| **:set ai** | 자동 들여쓰기 (auto indent) |
| :set noai | 자동 들여쓰기 해제 |
| **:set ts=n** | 탭 크기를 n으로 설정 (tab stop) |
| **:set sm** | 괄호 매칭 표시 (show match) |
| :set nosm | 괄호 매칭 해제 |
| :set ic | 검색 시 대소문자 무시 (ignore case) |
| :set noic | 대소문자 구분 검색 |
| :set paste | 붙여넣기 모드 (들여쓰기 방지) |
| **:n** | n번째 줄로 이동 |

### EX 모드 - 파일 관련

| 명령 | 설명 |
|------|------|
| :e 파일명 | 다른 파일 열기 (edit) |
| :r 파일명 | 현재 위치에 파일 내용 삽입 (read) |
| :! 명령어 | 셸 명령어 실행 |
| :sh | 셸로 전환 (exit로 복귀) |

## emacs 에디터

Richard Stallman이 개발한 다기능 에디터. 매크로 기능이 강력하고 확장성이 뛰어나다.

### emacs 주요 키

| 키 | 설명 |
|----|------|
| **Ctrl+x Ctrl+s** | **저장** (save) |
| **Ctrl+x Ctrl+c** | **종료** (close) |
| **Ctrl+k** | 커서부터 **줄 끝까지 삭제** (kill) |
| Ctrl+y | 삭제한 텍스트 붙여넣기 (yank) |
| Ctrl+a | 줄 처음으로 이동 |
| Ctrl+e | 줄 끝으로 이동 |
| Ctrl+f | 커서 한 칸 앞으로 (forward) |
| Ctrl+b | 커서 한 칸 뒤로 (backward) |
| Ctrl+n | 다음 줄 (next) |
| Ctrl+p | 이전 줄 (previous) |
| Ctrl+s | 검색 (search) |
| Ctrl+g | 명령 취소 |
| Ctrl+_ | 실행 취소 (undo) |
| Ctrl+x Ctrl+f | 파일 열기 (find-file) |
| Ctrl+x Ctrl+w | 다른 이름으로 저장 (write-file) |

> [!tip] emacs 키 기억법
> - **Ctrl+x Ctrl+s** = e**X**ecute + **S**ave
> - **Ctrl+x Ctrl+c** = e**X**ecute + **C**lose
> - **Ctrl+k** = **K**ill (줄 삭제)

## nano 에디터

초보자 친화적인 간단한 텍스트 에디터. 화면 하단에 사용 가능한 키가 표시된다.

### nano 주요 키

| 키 | 설명 |
|----|------|
| **Ctrl+O** | **저장** (Write Out) |
| **Ctrl+X** | **종료** (Exit) |
| **Ctrl+K** | 현재 줄 **잘라내기** (Cut) |
| Ctrl+U | 붙여넣기 (Uncut/Paste) |
| Ctrl+W | 검색 (Where Is) |
| Ctrl+\ | 검색/치환 (Replace) |
| Ctrl+G | 도움말 (Help) |
| Ctrl+C | 현재 커서 위치 표시 |
| Ctrl+_ | 특정 줄로 이동 (Go To Line) |
| Alt+U | 실행 취소 (Undo) |

> [!important] nano vs vi 비교
> | 구분 | nano | vi/vim |
> |------|------|--------|
> | 저장 | **Ctrl+O** | **:w** |
> | 종료 | **Ctrl+X** | **:q** |
> | 줄 삭제 | **Ctrl+K** | **dd** |
> | 난이도 | 쉬움 (키 표시) | 어려움 (모드 개념) |

## crontab 상세 (에디터 섹션)

vi 편집기를 사용하여 crontab을 편집하므로 에디터 단원에서도 중요하게 다룬다.

### crontab 형식

```
분  시  일  월  요일  명령어
┃   ┃   ┃   ┃   ┃
┃   ┃   ┃   ┃   ┗━ 요일 (0-7, 0과 7=일요일)
┃   ┃   ┃   ┗━━━ 월 (1-12)
┃   ┃   ┗━━━━━ 일 (1-31)
┃   ┗━━━━━━━ 시 (0-23)
┗━━━━━━━━━ 분 (0-59)
```

### 특수 문자

| 문자 | 의미 | 예시 |
|------|------|------|
| **\*** | **모든 값** | * = 매분/매시/매일 등 |
| **,** | **여러 값 나열** | 1,3,5 = 1, 3, 5 |
| **-** | **범위** | 1-5 = 1부터 5까지 |
| **/** | **간격** | */5 = 5마다 (5 간격) |

### 핵심 예시 ★★★

| crontab 항목 | 실행 시간 |
|-------------|----------|
| `0 13 * * *` | **매일 오후 1시** (13시 0분) |
| `0 0 5 * *` | **매달 5일 자정** (0시 0분) |
| `0 0 * * 1` | **매주 월요일 자정** (요일 1=월) |
| `*/5 * * * *` | **5분 간격**으로 (매시 0, 5, 10, ...) |
| `0 9 * * 1-5` | **월~금 오전 9시** |
| `30 */2 * * *` | **2시간마다 30분**에 (0:30, 2:30, 4:30, ...) |
| `0 0 1 1 *` | **매년 1월 1일 자정** |
| `0 6,18 * * *` | **매일 오전 6시, 오후 6시** |
| `0 0 * * 0` | **매주 일요일 자정** (0=일요일) |
| `*/10 9-17 * * *` | **오전 9시~오후 5시 사이 10분마다** |

### crontab 관리 명령어

```bash
crontab -e               # cron 작업 편집 (vi 편집기 사용)
crontab -l               # 현재 설정된 cron 목록 확인
crontab -r               # 모든 cron 작업 삭제 (주의!)
crontab -u username -l   # 특정 사용자의 cron 목록 (root만)
```

> [!warning] crontab -r 주의
> `crontab -r`은 **모든 cron 작업을 삭제**한다.
> 특정 작업만 삭제하려면 `crontab -e`로 편집하여 해당 줄을 삭제한다.
