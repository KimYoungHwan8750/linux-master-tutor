# 에디터 (★★)

#editor #linux-master

## Overview Table (한눈에 비교)
| 에디터 | 특징 | 모드 |
|--------|------|------|
| vi/vim | 리눅스 기본 에디터 | 명령/입력/라인 모드 |
| emacs | 확장 가능, 다기능 | 없음 (단일 모드) |
| nano | 간단, 초보 친화 | 없음 (단일 모드) |

## vi/vim 모드

```
          i, a, o
명령모드 ──────→ 입력모드
    ↑               │
    │     ESC        │
    ←────────────────┘
    │
    │  :
    ↓
 라인모드(ex모드)
```

| 모드 | 진입 방법 | 용도 |
|------|-----------|------|
| **명령모드** | ESC | 커서 이동, 복사, 삭제 |
| **입력모드** | i, a, o, I, A, O | 텍스트 입력 |
| **라인모드** | : | 저장, 종료, 검색, 치환 |

## 명령모드 핵심 키

### 커서 이동
| 키 | 이동 |
|----|------|
| h, j, k, l | 좌, 하, 상, 우 |
| w | 다음 단어 |
| b | 이전 단어 |
| 0 | 줄 맨 앞 |
| $ | 줄 맨 끝 |
| G | 파일 맨 끝 |
| gg | 파일 맨 앞 |

### 편집
| 키 | 기능 |
|----|------|
| dd | 줄 삭제 |
| Ndd | N줄 삭제 |
| yy | 줄 복사 |
| Nyy | N줄 복사 |
| p | 아래에 붙여넣기 |
| P | 위에 붙여넣기 |
| x | 문자 삭제 |
| u | 실행 취소 |
| Ctrl+R | 다시 실행 |

### 입력모드 진입
| 키 | 위치 |
|----|------|
| i | 커서 앞 |
| a | 커서 뒤 |
| o | 아래 줄 |
| I | 줄 맨 앞 |
| A | 줄 맨 끝 |
| O | 위 줄 |

## 라인모드 명령

| 명령 | 기능 |
|------|------|
| `:w` | 저장 |
| `:q` | 종료 |
| `:wq` | 저장 후 종료 |
| `:q!` | 저장 안 하고 종료 |
| `:wq!` | 강제 저장 후 종료 |
| `/text` | 아래로 검색 |
| `?text` | 위로 검색 |
| `n` | 다음 검색 결과 |
| `N` | 이전 검색 결과 |
| `:%s/old/new/g` | 전체 치환 |
| `:set nu` | 줄 번호 표시 |

## emacs

- GNU 프로젝트의 확장 가능한 에디터
- **Ctrl** 키 조합으로 대부분의 기능 수행
- Ctrl+x Ctrl+s = 저장, Ctrl+x Ctrl+c = 종료
- 모드 구분 없이 바로 편집 가능

## nano

- 간단하고 직관적인 에디터
- 화면 하단에 단축키 가이드 표시
- Ctrl+O = 저장, Ctrl+X = 종료
- 초보자에게 추천
