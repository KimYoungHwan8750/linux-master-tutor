#dashboard #linux-master

## 파일 시스템 → [[파일 권한과 소유권]], [[디스크 쿼타와 파일시스템]], [[파일시스템 관리 명령어]]

| 명령어 | 용도 | 주요 옵션 |
|--------|------|-----------|
| `ls -l` | 파일 상세 목록 | `-a` 숨김, `-R` 재귀 |
| `chmod` | 허가권 변경 | `755`, `u+x`, `4755`(SetUID) |
| `chown` | 소유자 변경 | `user:group file` |
| `chgrp` | 그룹 변경 | `group file` |
| `mount` | 장치 마운트 | `-t type device dir` |
| `umount` | 마운트 해제 | `device` or `dir` |
| `fdisk` | 파티션 관리 | `-l` 목록 |
| `mkfs` | 파일시스템 생성 | `-t ext4 device` |

### 특수 권한 요약
| 권한 | 숫자 | 위치 | 효과 |
|------|------|------|------|
| SetUID | 4000 | 소유자 x→s | 실행 시 소유자 권한 |
| SetGID | 2000 | 그룹 x→s | 실행 시 그룹 권한 |
| Sticky bit | 1000 | 기타 x→t | 소유자/root만 삭제 |

### 디스크 쿼타 명령어

| `quota` | 쿼타 확인 |

| `setquota` | CLI로 쿼타 설정 |
| `xfs_quota` | XFS 전용 쿼타 |
| `repquota` | 쿼타 보고서 |

## 에디터 → [[에디터]]

### vi 모드 전환
```
명령모드 ←(ESC)→ 입력모드(i,a,o)
    ↕ (:)
  라인모드
```

### vi 핵심 키
| 키 | 모드 | 기능 |
|----|------|------|
| `i` | 명령→입력 | 커서 앞 삽입 |
| `a` | 명령→입력 | 커서 뒤 삽입 |
| `o` | 명령→입력 | 아래 줄 삽입 |
| `dd` | 명령 | 줄 삭제 |
| `yy` | 명령 | 줄 복사 |
| `p` | 명령 | 붙여넣기 |
| `/text` | 명령 | 검색 |
| `:wq` | 라인 | 저장 후 종료 |
| `:q!` | 라인 | 저장 안 하고 종료 |
| `:%s/old/new/g` | 라인 | 전체 치환 |

## 네트워크 → [[네트워크 기초]], [[네트워크 명령어와 서비스]], [[클러스터링과 스토리지]]

### OSI 7계층 핵심
| 계층 | 이름 | 프로토콜/장비 |
|------|------|---------------|
| 7 | 응용 | HTTP, FTP, SMTP, DNS |
| 4 | 전송 | TCP, UDP |
| 3 | 네트워크 | IP, ICMP, ARP, RARP |
| 2 | 데이터링크 | 이더넷 |
| 1 | 물리 | 케이블 |

### TCP vs UDP
| 항목 | TCP | UDP |
|------|-----|-----|
| 연결 | 연결지향(3-way) | 비연결 |
| 신뢰성 | 높음 | 낮음 |
| 속도 | 느림 | 빠름 |
| 용도 | HTTP, FTP | DNS, 스트리밍 |

### 네트워크 명령어

| `ifconfig` | 네트워크 인터페이스 설정 |

### RAID 비교
| RAID | 방식 | 최소 디스크 | 특징 |
|------|------|-------------|------|
| 0 | 스트라이핑 | 2 | 성능↑, 안전성✗ |
| 1 | 미러링 | 2 | 안전성↑, 용량 50% |
| 5 | 패리티 분산 | 3 | 성능+안전성 균형 |

### LVM 구조
```
Physical Volume(PV) → Volume Group(VG) → Logical Volume(LV)
```

## Must-Know 공식/패턴
- 허가권 숫자: r=4, w=2, x=1 → `chmod 755` = rwxr-xr-x → [[파일 권한과 소유권]]
- crontab: `분 시 일 월 요일 명령` → [[프로세스 관리]]
- 소스 설치: `configure → make → make install` → [[소프트웨어 설치 및 삭제]]
- TCP 3-way: `SYN → SYN+ACK → ACK` → [[네트워크 기초]]
- LVM: `PV → VG → LV` → [[클러스터링과 스토리지]]
