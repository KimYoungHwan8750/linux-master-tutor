## 파일시스템 — 권한 → [[파일 권한과 소유권]]

| 항목 | 내용 |
|------|------|
| 읽기(r) | 4 |
| 쓰기(w) | 2 |
| 실행(x) | 1 |
| SetUID | 4000 (`s` on owner-x) |
| SetGID | 2000 (`s` on group-x) |
| Sticky Bit | 1000 (`t` on other-x) |

| umask 디렉토리 | 777 − umask |

## 인터넷 — 네트워크 기초 → [[네트워크 기초]]

### OSI 7계층

| 계층 | 이름 | 단위 | 프로토콜/장비 |
|------|------|------|---------------|
| 1 | 물리 | bit | 허브, 리피터 |
| 2 | 데이터링크 | frame | 스위치, 브리지 |
| 3 | 네트워크 | packet | 라우터, IP, ICMP, ARP |
| 4 | 전송 | segment | TCP, UDP |
| 5 | 세션 | - | NetBIOS, RPC |
| 6 | 표현 | - | JPEG, MPEG, SSL |
| 7 | 응용 | - | HTTP, FTP, SMTP, DNS |

### TCP vs UDP

| 항목 | TCP | UDP |
|------|-----|-----|
| 연결 | 연결 지향 | 비연결 |
| 신뢰성 | 높음 | 낮음 |
| 속도 | 느림 | 빠름 |
| 용도 | HTTP, FTP, SMTP | DNS, DHCP, SNMP |

### 주요 포트

| 포트 | 서비스 | 프로토콜 |
|------|--------|----------|
| 20/21 | FTP (데이터/제어) | TCP |
| 22 | SSH | TCP |
| 23 | Telnet | TCP |
| 25 | SMTP | TCP |
| 53 | DNS | TCP/UDP |
| 80 | HTTP | TCP |
| 110 | POP3 | TCP |
| 143 | IMAP | TCP |
| 443 | HTTPS | TCP |

### 주소 변환

| 프로토콜 | 방향 |
|----------|------|
| ARP | IP → MAC |
| RARP | MAC → IP |

## 인터넷 — 클러스터/스토리지 → [[클러스터링과 스토리지]]

### RAID

| 레벨 | 방식 | 최소 디스크 | 복구 |
|------|------|-------------|------|
| RAID 0 | 스트라이핑 | 2 | 불가 |
| RAID 1 | 미러링 | 2 | 가능 |
| RAID 5 | 스트라이핑 + 패리티 | 3 | 1개 장애 허용 |
| RAID 6 | 스트라이핑 + 이중 패리티 | 4 | 2개 장애 허용 |
| RAID 1+0 | 미러링 + 스트라이핑 | 4 | 가능 |

### LVM

| 구성 요소 | 약자 | 설명 |
|-----------|------|------|
| Physical Volume | PV | 물리적 디스크/파티션 |
| Volume Group | VG | PV의 그룹 |
| Logical Volume | LV | VG에서 할당된 논리 볼륨 |

> 순서: **PV → VG → LV**

| 명령 | 설명 |
|------|------|
| `pvcreate` / `pvdisplay` | PV 생성/조회 |
| `vgcreate` / `vgdisplay` | VG 생성/조회 |
| `lvcreate` / `lvdisplay` | LV 생성/조회 |

### 클러스터링

| 유형 | 목적 |
|------|------|
| HA (고가용성) | 장애 시 자동 전환 |
| HPC (고성능) | 병렬 처리 (Beowulf) |
| LVS (부하분산) | 트래픽 분배 |

### 클라우드 서비스 모델

| 모델 | 제공 범위 | 예시 |
|------|-----------|------|
| IaaS | 하드웨어 (인프라) | AWS EC2, OpenStack |
| PaaS | 플랫폼 (개발환경) | Heroku, Google App Engine |
| SaaS | 애플리케이션 | Google Docs, Office 365 |

### 가상화

| 기술 | 설명 |
|------|------|
| 전가상화 | 하드웨어 완전 가상화 (KVM) |
| 반가상화 | 게스트 OS 수정 필요 (Xen) |
| 컨테이너 | OS 수준 가상화 (Docker, LXC) |
