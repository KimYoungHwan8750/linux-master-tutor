### 본셸(Bourne Shell) 계열

| 셸 | 경로 | 개발자 | 특징 |
|----|------|--------|------|
| 본셸 (sh) | /bin/sh | Stephen Bourne (Bell Labs) | 최초의 유닉스 셸, 스크립트 기본 셸 |
| 콘셸 (ksh) | /bin/ksh | David Korn (AT&T Bell Labs) | 본셸 호환 + C셸 기능 결합, 히스토리/별칭 지원 |
| 배쉬셸 (bash) | /bin/bash | Brian Fox (GNU 프로젝트) | **POSIX 호환**, GNU 표준셸, 대부분 리눅스 기본셸 |
| 지셸 (zsh) | /bin/zsh | Paul Falstad | 강력한 히스토리, 자동완성, 플러그인, 테마 지원 |

## 각 셸 상세

### 본셸 (Bourne Shell, /bin/sh)

- **최초의 유닉스 셸** (1977년, Bell Labs)
- Stephen Bourne 개발
- 셸 스크립트의 기본 셸로 사용
- 히스토리, 별칭(alias) 기능 **미지원**
- 프롬프트: `$` (일반 사용자), `#` (root)

### 콘셸 (Korn Shell, /bin/ksh)

- David Korn 개발 (AT&T Bell Labs)
- **본셸과 호환**되면서 C셸의 기능을 결합
- 히스토리, 별칭(alias), 작업 제어 지원
- 명령행 편집 기능 지원

### 배쉬셸 (Bash, /bin/bash) ★

- Brian Fox 개발 (GNU 프로젝트)
- **Bourne Again Shell**의 약자
- **POSIX 호환** (표준 준수)
- 대부분의 리눅스 배포판 **기본 셸**
- 본셸, 콘셸의 기능을 포함 + C셸의 기능도 일부 포함
- 히스토리, 별칭, 명령행 편집, 프로그래밍, 자동완성 모두 지원
- Tab 키로 명령어/파일명 자동완성

### 지셸 (Z Shell, /bin/zsh)

- Paul Falstad 개발
- **강력한 히스토리 기능** (공유 히스토리)
- 강력한 자동완성 (경로, 옵션까지)
- 플러그인, 테마 시스템 (Oh My Zsh 등)
- macOS Catalina 이후 기본 셸

### C셸 (C Shell, /bin/csh)

- Bill Joy 개발 (UC Berkeley)
- **C 언어 문법**을 기반으로 한 셸
- **히스토리, 별칭(alias) 기능을 최초로 도입**
- 프롬프트: `%`
- 셸 스크립트 문법이 본셸과 다름

### tc셸 (tcsh, /bin/tcsh)

- C셸을 개선한 셸
- **명령어 자동완성(Tab 키)** 기능 추가
- 명령행 편집 기능 향상
- C셸과 완전 호환

## /etc/shells 파일

시스템에서 사용 가능한 **셸 목록**을 저장하는 파일.

```bash
cat /etc/shells
```

출력 예:
```
/bin/sh
/bin/bash
/bin/csh
/bin/tcsh
/bin/ksh
/bin/zsh
```

> [!tip] 참고
> 새로운 셸을 설치한 후 `/etc/shells`에 등록해야 chsh로 변경 가능하다.

## 셸 변경 방법

### chsh (Change Shell)

사용자의 **로그인 셸을 변경**한다.

```bash
chsh [옵션] [사용자명]
chsh -s /bin/zsh          # 현재 사용자의 셸을 zsh로 변경
chsh -s /bin/bash june    # june 사용자의 셸을 bash로 변경
chsh -l                   # 사용 가능한 셸 목록 (/etc/shells 내용)
```

| 옵션 | 설명 |
|------|------|
| -s 셸경로 | 변경할 셸 지정 |
| -l | 사용 가능한 셸 목록 표시 |

### usermod 명령어로 셸 변경

```bash
usermod -s /bin/zsh june    # june 사용자의 셸을 zsh로 변경
```

> [!important] chsh vs usermod
> - **chsh -s**: 셸 변경 전용 명령어
> - **usermod -s**: 사용자 정보 변경 명령어의 셸 변경 옵션

---

## 현재 셸 확인

```bash
echo $SHELL        # 현재 로그인 셸 확인
cat /etc/passwd    # 사용자별 기본 셸 확인 (마지막 필드)
```

**/etc/passwd 형식**:
```
사용자명:비밀번호:UID:GID:설명:홈디렉토리:로그인셸
june:x:1000:1000:June User:/home/june:/bin/bash
```

마지막 필드(`/bin/bash`)가 해당 사용자의 기본 로그인 셸이다.
