# 셸의 종류와 특징 (★★★)

#shell #linux-master

## Overview Table (한눈에 비교)
| 셸 | 계열 | 실행파일 | 특징 |
|----|------|----------|------|
| sh | 본셸 | /bin/sh | 최초의 유닉스 셸 |
| ksh | 본셸 | /bin/ksh | 히스토리, alias, 작업 제어 |
| bash | 본셸 | /bin/bash | 리눅스 기본 셸, sh 호환 |
| zsh | 본셸 | /bin/zsh | 강력한 자동완성, 플러그인 |
| csh | C셸 | /bin/csh | C언어 문법 기반 |
| tcsh | C셸 | /bin/tcsh | csh 개선판, 명령줄 편집 |

## 본셸(Bourne Shell) 계열

- **sh**: Stephen Bourne이 개발한 최초의 셸
- **bash**: **B**ourne **A**gain **Sh**ell — GNU 프로젝트, 리눅스 표준 셸
- **ksh**: Korn Shell — 히스토리, alias 기능 추가
- **zsh**: Z Shell — bash 호환 + 강력한 자동완성

## C셸 계열

- **csh**: C언어 문법 기반, 히스토리/alias 최초 도입
- **tcsh**: csh에 명령줄 편집, 자동완성 추가

## 셸 관련 명령어

| 명령어 | 기능 |
|--------|------|
| `echo $SHELL` | 현재 로그인 셸 확인 |
| `cat /etc/shells` | 사용 가능한 셸 목록 |
| `chsh` | 로그인 셸 변경 |
| `chsh -s /bin/zsh` | zsh로 변경 |
| `/etc/passwd` | 사용자별 로그인 셸 확인 (마지막 필드) |

> [!important] chsh 주의사항
> `/etc/shells`에 등록된 셸만 변경 가능. 등록되지 않은 셸로 변경 시도 시 에러 발생.
