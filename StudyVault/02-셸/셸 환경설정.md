# 셸 환경설정 (★★★)

#shell #linux-master

## Overview Table (한눈에 비교)
| 파일 | 범위 | 용도 | 실행 시점 |
|------|------|------|-----------|
| `/etc/profile` | 전체 사용자 | 환경변수, PATH | 로그인 시 |
| `/etc/bashrc` | 전체 사용자 | alias, 함수 | 셸 시작 시 |
| `~/.bash_profile` | 개인 | 환경변수 | 로그인 시 |
| `~/.bashrc` | 개인 | alias, 함수 | 비로그인 셸 시작 시 |
| `~/.bash_logout` | 개인 | 정리 작업 | 로그아웃 시 |

## 환경설정 파일 로드 순서

```
로그인 셸:
/etc/profile → ~/.bash_profile → ~/.bashrc → /etc/bashrc

비로그인 셸:
~/.bashrc → /etc/bashrc
```

> [!tip] 전체 vs 개인 구분법
> - `/etc/` 경로 = **전체** 사용자 적용
> - `~/` (홈 디렉토리) = **개인** 사용자만 적용

## history 명령어

| 명령어 | 기능 |
|--------|------|
| `history` | 명령 히스토리 출력 |
| `history N` | 최근 N개 출력 |
| `!!` | 직전 명령 재실행 |
| `!N` | N번째 명령 재실행 |
| `!string` | string으로 시작하는 최근 명령 재실행 |
| `HISTSIZE` | 메모리 히스토리 크기 |
| `HISTFILE` | 히스토리 저장 파일 (~/.bash_history) |
| `HISTFILESIZE` | 파일 히스토리 크기 |

## alias

```bash
alias ll='ls -la'       # alias 설정
alias                    # 현재 alias 목록
unalias ll               # alias 해제
```

- 영구 적용: `~/.bashrc`에 추가

## 주요 환경변수

| 변수 | 용도 |
|------|------|
| `$HOME` | 홈 디렉토리 경로 |
| `$PATH` | 명령어 검색 경로 |
| `$SHELL` | 현재 셸 |
| `$USER` | 현재 사용자 |
| `$PS1` | 프롬프트 형식 |
| `$LANG` | 언어 설정 |
