# 셸 환경설정 (★★★)

#linux-master #shell #env-var #config-file

## 한눈에 비교

| 명령어/파일 | 기능 | 핵심 포인트 |
|------------|------|------------|
| env | 환경변수 목록 표시 | 현재 셸의 환경변수 출력 |
| set | 셸 변수 + 환경변수 모두 표시 | 모든 변수 출력 |
| export | 셸 변수를 환경변수로 전환 | 자식 프로세스에 전달 |
| printenv | 환경변수 값 표시 | 특정 변수 값 확인 |
| /etc/profile | 시스템 전체, 로그인 시 | 전역 설정 |
| /etc/bashrc | 시스템 전체, bash 실행 시 | 전역 bash 설정 |
| ~/.bash_profile | 개인, 로그인 시 | 사용자별 로그인 설정 |
| ~/.bashrc | 개인, bash 실행 시 | 사용자별 bash 설정 |

## 환경변수 (Environment Variables)

### 주요 환경변수

| 변수명 | 설명 | 예시 |
|--------|------|------|
| **PATH** | 명령어 검색 경로 | /usr/local/bin:/usr/bin:/bin |
| **HOME** | 사용자 홈 디렉토리 | /home/june |
| **SHELL** | 현재 로그인 셸 | /bin/bash |
| **USER** | 현재 사용자명 | june |
| **HOSTNAME** | 시스템 호스트명 | myserver |
| **LANG** | 언어/로케일 설정 | ko_KR.UTF-8 |
| **PS1** | 기본 프롬프트 문자열 | [\u@\h \w]\$ |
| **PS2** | 보조 프롬프트 (명령행 연속) | > |
| **HISTSIZE** | 히스토리 저장 명령어 수 | 1000 |
| **HISTFILE** | 히스토리 저장 파일 경로 | ~/.bash_history |
| **DISPLAY** | X 윈도우 디스플레이 서버 | :0.0 |
| **TERM** | 터미널 유형 | xterm-256color |
| **LOGNAME** | 로그인 사용자명 | june |
| **PWD** | 현재 작업 디렉토리 | /home/june |
| **MAIL** | 메일 저장 경로 | /var/spool/mail/june |

### 환경변수 확인 방법

```bash
echo $PATH          # 특정 환경변수 값 출력
echo $HOME          # 홈 디렉토리 확인
env                 # 모든 환경변수 목록
printenv PATH       # 특정 환경변수 값 확인
set                 # 셸 변수 + 환경변수 모두 출력
```

### 환경변수 설정/해제

```bash
# 셸 변수 설정 (현재 셸에서만 유효)
MY_VAR="hello"

# 환경변수로 전환 (자식 프로세스에도 전달)
export MY_VAR="hello"
export PATH=$PATH:/new/path    # PATH에 경로 추가

# 변수 해제
unset MY_VAR
```

> [!important] 셸 변수 vs 환경변수
> | 구분 | 셸 변수 | 환경변수 |
> |------|---------|---------|
> | 설정 | `VAR=value` | `export VAR=value` |
> | 범위 | **현재 셸**에서만 유효 | **자식 프로세스**에도 전달 |
> | 확인 | set | env, printenv |

## PS1 / PS2 프롬프트

### PS1 (기본 프롬프트)

셸에서 명령어를 입력받을 때 표시되는 프롬프트 문자열.

**PS1 특수문자**:

| 특수문자 | 설명 | 예시 출력 |
|----------|------|----------|
| **\u** | 현재 **사용자**명 | june |
| **\h** | **호스트**명 (첫 번째 . 이전) | myserver |
| **\H** | 전체 호스트명 (FQDN) | myserver.example.com |
| **\w** | 현재 **작업 디렉토리** (전체 경로) | ~/projects |
| **\W** | 현재 작업 디렉토리 (마지막 디렉토리만) | projects |
| **\$** | **프롬프트** 기호 (root=#, 일반=$) | $ 또는 # |
| \d | 날짜 (요일 월 일) | Mon Oct 01 |
| \t | 시간 (24시간, HH:MM:SS) | 14:30:45 |
| \T | 시간 (12시간, HH:MM:SS) | 02:30:45 |
| \n | 줄바꿈 | |
| \# | 명령어 번호 | 42 |

```bash
# 기본 PS1 설정 예
PS1='[\u@\h \w]\$ '
# 출력: [june@myserver ~/projects]$

# 사용자 정의 예
PS1='\u:\w> '
# 출력: june:~/projects>
```

> [!tip] 시험에서 자주 묻는 PS1 특수문자
> - `\u` = **사용자(user)**
> - `\h` = **호스트(host)**
> - `\w` = **작업 디렉토리(working directory)**
> - `\$` = **프롬프트 기호** (root이면 #, 일반이면 $)

### PS2 (보조 프롬프트)

명령이 한 줄에서 완성되지 않아 **다음 줄로 이어질 때** 표시되는 프롬프트.
기본값: `>`

```bash
$ echo "hello \
> world"           # > 가 PS2 프롬프트
hello world

$ PS2='... '       # PS2를 ...으로 변경
$ echo "hello \
... world"
```

## 셸 설정 파일

### 설정 파일 적용 범위와 시점

```
시스템 전체 (모든 사용자)          개인 (해당 사용자만)
┌──────────────────┐          ┌──────────────────┐
│  /etc/profile    │ 로그인 시 │ ~/.bash_profile  │ 로그인 시
│  (전역 로그인)    │          │ (개인 로그인)     │
├──────────────────┤          ├──────────────────┤
│  /etc/bashrc     │ bash 시  │ ~/.bashrc        │ bash 실행 시
│  (전역 bash)     │          │ (개인 bash)       │
└──────────────────┘          └──────────────────┘
```

| 설정 파일 | 적용 범위 | 적용 시점 | 설명 |
|-----------|----------|----------|------|
| **/etc/profile** | **시스템 전체** | **로그인 시** | 모든 사용자에게 적용되는 전역 환경 설정 |
| **/etc/bashrc** | **시스템 전체** | **bash 실행 시** | 모든 사용자의 bash 실행 시 적용 |
| **~/.bash_profile** | **개인** | **로그인 시** | 해당 사용자만 적용, 로그인 시 한 번 실행 |
| **~/.bashrc** | **개인** | **bash 실행 시** | 해당 사용자만 적용, bash 시작할 때마다 실행 |
| ~/.bash_logout | 개인 | **로그아웃 시** | 로그아웃 시 실행할 명령어 |
| ~/.bash_history | 개인 | - | 명령어 히스토리 저장 파일 |

> [!important] 적용 순서
> 로그인 시: `/etc/profile` → `~/.bash_profile` → `~/.bashrc` → `/etc/bashrc`

### 로그인 셸 vs 비로그인 셸

| 구분 | 로그인 셸 | 비로그인 셸 |
|------|----------|------------|
| 상황 | 시스템 로그인, ssh 접속 | 터미널 새 창, bash 명령 실행 |
| 읽는 파일 | /etc/profile → ~/.bash_profile | ~/.bashrc → /etc/bashrc |

## 명령어 상세

### env

현재 셸의 **환경변수 목록**을 출력한다.

```bash
env                     # 모든 환경변수 출력
env | grep PATH         # PATH 관련 변수 검색
```

### set

**셸 변수와 환경변수 모두**를 출력한다 (env보다 많은 정보).

```bash
set                     # 모든 변수 출력
set | grep MY_VAR       # 특정 변수 검색
```

### export

셸 변수를 **환경변수로 전환**하여 자식 프로세스에서도 사용할 수 있게 한다.

```bash
export VAR="value"       # 환경변수 설정
export PATH=$PATH:/opt/bin  # PATH에 경로 추가
export                   # export된 변수 목록 표시
```

### printenv

환경변수의 값을 출력한다.

```bash
printenv                 # 모든 환경변수 출력
printenv PATH            # PATH 값만 출력
printenv HOME SHELL      # 여러 변수 값 출력
```

## history 명령어

이전에 실행한 명령어의 **히스토리 목록**을 관리한다.

```bash
history              # 전체 히스토리 출력
history 10           # 최근 10개 명령어
!!                   # 직전 명령어 재실행
!n                   # n번 히스토리 명령어 재실행
!문자열               # 해당 문자열로 시작하는 최근 명령어 재실행
history -c           # 히스토리 목록 삭제
history -w           # 히스토리를 파일에 저장
```

| 관련 변수 | 설명 |
|-----------|------|
| HISTSIZE | 메모리에 저장할 히스토리 수 |
| HISTFILESIZE | 히스토리 파일에 저장할 최대 줄 수 |
| HISTFILE | 히스토리 저장 파일 경로 (기본: ~/.bash_history) |

## alias / unalias

명령어에 **별칭(alias)**을 설정/해제한다.

```bash
alias                      # 현재 설정된 별칭 목록
alias ll='ls -al'          # ll을 ls -al로 설정
alias rm='rm -i'           # rm에 확인 옵션 추가
unalias ll                 # ll 별칭 해제
unalias -a                 # 모든 별칭 해제
```

> [!tip] 영구 alias 설정
> alias는 현재 세션에서만 유효하다. 영구적으로 설정하려면 `~/.bashrc`에 추가한다.

## /etc/passwd 필드 구조

각 사용자의 계정 정보를 저장하는 파일.

```
사용자명:비밀번호:UID:GID:설명(GECOS):홈디렉토리:로그인셸
```

| 필드 | 번호 | 설명 | 예시 |
|------|------|------|------|
| 사용자명 | 1 | 로그인 ID | june |
| 비밀번호 | 2 | x (실제는 /etc/shadow에 저장) | x |
| UID | 3 | 사용자 ID (root=0) | 1000 |
| GID | 4 | 기본 그룹 ID | 1000 |
| 설명 (GECOS) | 5 | 사용자 설명/이름 | June User |
| 홈 디렉토리 | 6 | 로그인 시 작업 디렉토리 | /home/june |
| 로그인 셸 | 7 | 기본 셸 | /bin/bash |

```bash
# 예시
root:x:0:0:root:/root:/bin/bash
june:x:1000:1000:June User:/home/june:/bin/bash
nobody:x:65534:65534:Nobody:/nonexistent:/usr/sbin/nologin
```

> [!important] 시험 포인트
> - root의 UID는 반드시 **0**
> - 비밀번호 필드의 **x**는 /etc/shadow에 암호화되어 저장됨을 의미
> - 마지막 필드가 `/usr/sbin/nologin` 또는 `/bin/false`면 로그인 불가능한 계정
