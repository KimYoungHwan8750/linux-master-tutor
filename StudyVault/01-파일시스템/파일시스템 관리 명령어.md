# 파일시스템 관리 명령어 (★★)

#filesystem #linux-master

## Overview Table (한눈에 비교)
| 명령어 | 기능 | 핵심 옵션 |
|--------|------|-----------|
| `mount` | 장치 마운트 | `-t type`, `-o options` |
| `umount` | 마운트 해제 | 장치명 또는 마운트포인트 |
| `fdisk` | 파티션 관리 | `-l` (목록) |
| `mkfs` | 파일시스템 생성 | `-t ext4` |
| `df` | 디스크 사용량 | `-h` (읽기 쉬운 단위) |
| `du` | 디렉토리 사용량 | `-sh` (합계) |

## mount / umount

장치를 디렉토리에 연결(마운트)하거나 해제하는 명령.

```
mount -t ext4 /dev/sdb1 /mnt/data
umount /mnt/data
```

- `/etc/fstab` — 부팅 시 자동 마운트 설정 파일
- `mount -a` — fstab에 정의된 모든 장치 마운트

### /etc/fstab 형식
```
장치명  마운트포인트  파일시스템  옵션  덤프  fsck순서
/dev/sda1  /  ext4  defaults  1  1
```

## fdisk / mkfs

| 명령어 | 역할 | 사용 예 |
|--------|------|---------|
| `fdisk /dev/sdb` | 파티션 생성/삭제/변경 | 대화형 |
| `fdisk -l` | 파티션 목록 확인 | |
| `mkfs -t ext4 /dev/sdb1` | 파일시스템 포맷 | |
| `mkfs.xfs /dev/sdb1` | XFS로 포맷 | |

> [!important] 작업 순서
> `fdisk`로 파티션 생성 → `mkfs`로 파일시스템 생성 → `mount`로 마운트

## df / du

| 명령어 | 용도 | 차이 |
|--------|------|------|
| `df` | 파일시스템 전체 사용량 | 마운트된 파일시스템 단위 |
| `du` | 특정 디렉토리 사용량 | 디렉토리/파일 단위 |

## Exam/Test Patterns (시험 빈출 패턴)
| Scenario/Keyword | Answer |
|-------------------|--------|
| "부팅 시 자동 마운트 파일" | **/etc/fstab** |
