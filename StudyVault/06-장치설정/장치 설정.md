## 프린터 설정

### CUPS (Common Unix Printing System)
- 현재 리눅스 **표준** 프린터 시스템
- 웹 기반 관리: `http://localhost:631`
- **IPP**(Internet Printing Protocol) 사용
- 설정 파일: `/etc/cups/cupsd.conf`

### LPRng
- 레거시 프린터 시스템
- BSD 계열 인쇄 시스템의 개선판

### 프린터 명령어
| 명령어 | 용도 |
|--------|------|
| `lpr` | 인쇄 요청 |
| `lpq` | 인쇄 큐 확인 |
| `lprm` | 인쇄 작업 취소 |
| `lpc` | 프린터 제어 |
| `lp` | System V 계열 인쇄 |
| `lpstat` | 프린터 상태 확인 |
| `cancel` | System V 계열 작업 취소 |

> [!tip] BSD vs System V 명령어
> - BSD 계열: lpr, lpq, lprm, lpc
> - System V 계열: lp, lpstat, cancel

## 사운드 카드

| 시스템 | 설명 |
|--------|------|
| **OSS** (Open Sound System) | 초기 리눅스 사운드 시스템, 레거시 |
| **ALSA** (Advanced Linux Sound Architecture) | 현재 표준, OSS 호환 모드 제공 |

- ALSA 관련 명령어: `alsamixer` (볼륨 조절), `aplay` (재생), `arecord` (녹음)

## 스캐너

| 시스템 | 역할 |
|--------|------|
| **SANE** (Scanner Access Now Easy) | 백엔드 (하드웨어 드라이버) |
| **XSANE** | 프론트엔드 (GUI 인터페이스) |

- `scanimage` — CLI에서 스캔 수행
- `sane-find-scanner` — 스캐너 장치 검색
