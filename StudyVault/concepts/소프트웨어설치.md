### 오답 메모

**APT (apt-get purge)**
- 문제: Debian 계열에서 패키지를 설정 파일까지 완전히 삭제하는 명령은?
- 내 답: apt-get remove apache2 (2회 연속 오답)
- 정답: apt-get purge apache2
- 핵심: remove=설정 유지, purge=설정 포함 완전 삭제. "설정까지"→purge, "설정 유지"→remove.

**apt-cache search / show**
- 문제: Debian 계열에서 패키지를 검색하는 명령은?
- 내 답: apt-get search web (2회 연속 오답)
- 정답: apt-cache search web
- 핵심: apt-get=동작(install/remove/purge/update/upgrade), apt-cache=조회(search/show). apt-get에는 search가 없음.
