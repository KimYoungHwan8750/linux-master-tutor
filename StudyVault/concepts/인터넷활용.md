### 오답 메모

**OSI 계층별 데이터 단위**
- 문제: 전송 계층(4계층)의 데이터 단위는?
- 내 답: Packet
- 정답: Segment
- 핵심: 1물리=Bit, 2데이터링크=Frame, 3네트워크=Packet, 4전송=Segment, 5~7=Data.

**ARP (Address Resolution Protocol)**
- 문제: IP 주소를 알고 MAC 주소를 모를 때 변환하는 프로토콜은?
- 내 답: 모름
- 정답: ARP
- 핵심: ARP=IP→MAC, RARP(Reverse ARP)=MAC→IP. 시험 빈출.

**LAN / MAN / WAN / SAN 비교**
- 문제: 데이터센터 내에서 스토리지 장치를 전용으로 연결하는 고속 네트워크는?
- 내 답: LAN
- 정답: SAN (Storage Area Network)
- 핵심: SAN=스토리지 전용 고속 네트워크. LAN=빌딩 내 일반 네트워크.

**토폴로지: 망형(Mesh)**
- 문제: 모든 장치가 일대일로 연결되어 신뢰성이 가장 높은 네트워크 토폴로지는?
- 내 답: 성형 (Star)
- 정답: 망형 (Mesh)
- 핵심: 망형=일대일 연결, 신뢰성 최고, 비용 높음. 성형=중앙 장치 고장 시 전체 마비.

**주요 서비스별 포트 번호**
- 문제: SMTP가 사용하는 기본 포트 번호는?
- 내 답: 53 (DNS)
- 정답: 25
- 핵심: FTP=21, SSH=22, Telnet=23, SMTP=25, DNS=53, HTTP=80, POP3=110, IMAP=143, HTTPS=443.

**레거시 vs 현대 네트워크 명령어 대응표**
- 문제: ifconfig를 대체하는 현대적 네트워크 설정 명령어는?
- 내 답: netstat
- 정답: ip
- 핵심: ifconfig→ip, route→ip route, netstat→ss, arp→ip neigh.

**Xen (반가상화) vs KVM (전가상화)**
- 문제: CPU 반가상화(Paravirtualization) 방식을 사용하는 가상화 솔루션은?
- 내 답: VirtualBox
- 정답: Xen
- 핵심: Xen=반가상화(게스트OS 수정 필요, 성능 우수), KVM=전가상화(레드햇, 커널 기반), VirtualBox=전가상화(오라클, 데스크톱용).

**LAN 기술 (FDDI)**
- 문제: 광섬유 기반의 이중 링 구조를 가진 LAN 기술은?
- 내 답: Ethernet
- 정답: FDDI (Fiber Distributed Data Interface)
- 핵심: FDDI=광섬유 링, Ethernet=유선 LAN(가장 보편적), Token Ring=토큰 기반.

**IP / ICMP / IGMP 프로토콜**
- 문제: IP 전달 오류를 보고하는 메시지 프로토콜로, ping이 사용하는 것은?
- 내 답: IP
- 정답: ICMP (Internet Control Message Protocol)
- 핵심: ICMP=오류 보고(ping), IGMP=멀티캐스트 그룹 관리, IP=기본 전송 규약.

**TCP 3-Way Handshake**
- 문제: 연결 설정 시 두 번째 단계에서 서버가 보내는 플래그는?
- 내 답: ACK
- 정답: SYN+ACK
- 핵심: 1단계 SYN → 2단계 SYN+ACK → 3단계 ACK.

**TCP 4-Way Handshake**
- 문제: 연결 해제 과정에서 첫 번째로 보내는 플래그는?
- 내 답: SYN
- 정답: FIN
- 핵심: SYN=연결 시작, FIN=종료 요청. 해제는 FIN부터.

**국제 기구**
- 문제: IP 주소 할당 및 도메인 관리를 총괄하는 국제 기구는?
- 내 답: ISO
- 정답: ICANN
- 핵심: ICANN=IP+도메인, IEEE=LAN 표준, ISO=OSI 모델, IETF=프로토콜(RFC), W3C=웹 표준.

**ip 명령어**
- 문제: ip 명령으로 ARP 테이블을 확인하는 서브커맨드는?
- 내 답: ip route
- 정답: ip neigh
- 핵심: ip addr=IP, ip link=인터페이스, ip route=라우팅, ip neigh=ARP(이웃).

**traceroute 명령어**
- 문제: 목적지까지의 패킷 경로를 추적하는 명령어는?
- 내 답: route
- 정답: traceroute
- 핵심: route=라우팅 테이블(정적), traceroute=실제 경로 추적(동적).

**route 명령어**
- 문제: 기본 게이트웨이를 추가하는 route 명령어는?
- 내 답: route add -net 10.0.0.0
- 정답: route add default gw IP
- 핵심: default gw=기본 게이트웨이, -net=특정 네트워크 라우트.

**arp 명령어**
- 문제: ARP 테이블에 정적 항목을 추가하는 명령은?
- 내 답: arp -a
- 정답: arp -s IP MAC
- 핵심: -a=모두 보기, -s=set(추가), -d=delete(삭제).

**Gopher**
- 문제: 미네소타 대학에서 개발한 터미널 기반 메뉴 구조 검색 시스템은?
- 내 답: 모름
- 정답: Gopher
- 핵심: 미네소타 대학 마스코트, 텍스트 기반, WWW 이전 서비스.

**전자 메일**
- 문제: 메일을 서버에 보관하며 여러 기기에서 동기화하여 수신하는 프로토콜은?
- 내 답: 모름
- 정답: IMAP (Internet Message Access Protocol)
- 핵심: SMTP(25)=송신, POP3(110)=다운로드 후 삭제, IMAP(143)=서버 보관 동기화.
