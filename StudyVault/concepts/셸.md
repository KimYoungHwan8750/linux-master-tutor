### 오답 메모

**~/.bash_profile**
- 문제: SSH로 원격 접속할 때 사용자의 개인 설정이 로드되는 파일은?
- 내 답: ~/.profile
- 정답: ~/.bash_profile
- 핵심: bash에서 SSH=로그인 셸→~/.bash_profile. ~/.profile은 bash_profile이 없을 때 대체.

**로그인 셸 vs 비로그인 셸**
- 문제: 'su -' 명령어로 사용자를 전환하면 어떤 유형의 셸이 시작되는가?
- 내 답: 모름
- 정답: 로그인 셸
- 핵심: su -=로그인 셸(환경 초기화), su=비로그인 셸(환경 유지). 하이픈(-)이 핵심.

**설정 파일 적용 순서**
- 문제: bash 로그인 셸이 시작될 때 가장 먼저 읽히는 설정 파일은?
- 내 답: ~/.bashrc
- 정답: /etc/profile
- 핵심: 로그인 셸 순서: /etc/profile → ~/.bash_profile → ~/.bashrc → /etc/bashrc. 시스템 전역이 항상 먼저.
