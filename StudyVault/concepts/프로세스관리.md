### 오답 메모

**데몬(Daemon) 프로세스**
- 문제: 클라이언트 요청이 있을 때만 프로세스를 생성하여 메모리를 효율적으로 사용하는 데몬 동작 방식은?
- 내 답: systemd
- 정답: xinetd
- 핵심: standalone=메모리 상주/즉시 응답, xinetd(eXtended Internet Services Daemon)=요청 시 생성/메모리 효율적. systemd는 시스템/서비스 관리자로 데몬 동작 방식이 아님.

**ps 명령어와 주요 옵션**
- 문제: BSD 스타일로 모든 프로세스를 사용자 중심 포맷으로 출력하는 명령어는?
- 내 답: ps -ef
- 정답: ps aux
- 핵심: ps aux=BSD 스타일(USER,%CPU,%MEM 표시), ps -ef=System V 스타일(UID,PPID 표시). aux는 BSD, -ef는 System V.

**kill 명령어**
- 문제: kill 명령어에서 시그널을 지정하지 않았을 때 기본으로 전송되는 시그널은?
- 내 답: SIGKILL (9)
- 정답: SIGTERM (15)
- 핵심: kill 기본값=SIGTERM(15) 정상 종료 요청. SIGKILL(9)은 -9를 명시해야 하는 강제 종료.

**killall / pkill 명령어**
- 문제: httpd라는 이름의 프로세스를 모두 종료하려면 어떤 명령어를 사용하는가?
- 내 답: kill httpd
- 정답: killall httpd
- 핵심: kill은 PID(숫자)로만 시그널 전송. 프로세스 이름으로 종료하려면 killall 또는 pkill 사용.

**nice 명령어 (NI값 설정)**
- 문제: 프로세스를 시작할 때 nice 값 10으로 실행하는 올바른 명령어는?
- 내 답: 모름
- 정답: nice -n 10 ./script.sh
- 핵심: nice -n 값 명령어 형식. nice=시작 시 지정, renice=실행 중 변경.

**NI값 범위**
- 문제: 일반 사용자(root가 아닌)가 설정할 수 있는 nice 값의 범위는?
- 내 답: 모름
- 정답: 0 ~ 19
- 핵심: 전체 범위 -20~19, 일반 사용자는 0~19(양수만). 음수(우선순위 높이기)는 root만 가능.

**crontab 필드 형식**
- 문제: crontab에서 "매주 월요일 자정"에 실행하도록 설정하는 시간 필드는?
- 내 답: 모름
- 정답: 0 0 * * 1
- 핵심: 필드 순서=분 시 일 월 요일. 요일: 0=일, 1=월, ..., 6=토, 7=일.

**포그라운드 vs 백그라운드 프로세스**
- 문제: 포그라운드에서 실행 중인 프로세스를 일시 정지시키는 키 조합과 해당 시그널은?
- 내 답: 모름
- 정답: Ctrl+Z / SIGTSTP(20)
- 핵심: Ctrl+C=종료(SIGINT/2), Ctrl+Z=일시 정지(SIGTSTP/20). 정지 후 bg/fg로 전환 가능.
