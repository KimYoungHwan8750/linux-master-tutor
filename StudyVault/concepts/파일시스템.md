### 오답 메모

**SetUID**
- 문제: passwd 명령어에 SetUID가 설정되어 있을 때, 일반 사용자가 비밀번호를 변경하면 어떤 권한으로 접근하는가?
- 내 답: 모름
- 정답: root(소유자) 권한
- 핵심: SetUID=실행 시 파일 소유자 권한으로 동작. passwd 소유자=root이므로 root 권한으로 /etc/shadow 수정.

**SetGID**
- 문제: 디렉토리에 SetGID(chmod 2755)를 설정하면 새로 생성되는 파일의 그룹 소유권은?
- 내 답: 파일 생성자의 기본 그룹
- 정답: 디렉토리의 그룹을 상속
- 핵심: SetGID on 파일=그룹 권한 실행, SetGID on 디렉토리=하위 파일이 디렉토리 그룹 상속.

**특수 권한 표기 (s/S, t/T)**
- 문제: ls -l 출력이 '-rwSr--r--'일 때 대문자 S가 의미하는 것은?
- 내 답: Sticky Bit 설정
- 정답: SetUID 설정, 소유자 실행 권한 없음
- 핵심: 소유자 자리=SetUID, 그룹 자리=SetGID, 기타 자리=Sticky Bit. 실행권한 있으면 소문자(s,t), 없으면 대문자(S,T).

**UUID와 blkid**
- 문제: 장치의 UUID, 파일시스템 유형, 레이블을 확인하는 명령어는?
- 내 답: fdisk -l
- 정답: blkid
- 핵심: blkid=UUID/TYPE/LABEL 식별, fdisk -l=파티션 테이블(크기/섹터). 2회 연속 오답.

**fstab defaults 옵션**
- 문제: /etc/fstab의 defaults에 포함되지 않는 것은?
- 내 답: 모름
- 정답: usrquota (defaults = rw,suid,dev,exec,auto,nouser,async)
- 핵심: defaults 7개 암기 필수. usrquota는 별도 명시 필요.
