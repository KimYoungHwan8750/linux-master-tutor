### 오답 메모

**chgrp 명령어**
- 문제: 파일의 그룹 소유권만 변경하는 전용 명령어는?
- 내 답: chown -g
- 정답: chgrp
- 핵심: 그룹 소유권만 변경하는 전용 명령어는 chgrp. chown -g 옵션은 존재하지 않음.

**SetUID**
- 문제: SetUID를 설정하여 실행 파일에 소유자 권한을 부여하는 chmod 명령어는?
- 내 답: chmod 1755 (1000)
- 정답: chmod 4755 (4000)
- 핵심: SetUID=4000, SetGID=2000, Sticky Bit=1000

**SetGID**
- 문제: SetGID의 8진수 특수 권한 값은?
- 내 답: 모름
- 정답: 2000
- 핵심: SetGID=2000, 그룹 실행 권한 자리에 표기

**특수 권한 표기**
- 문제: 특수 권한에서 소문자 s와 대문자 S의 차이는?
- 내 답: 모름
- 정답: 실행 권한 유무에 따라 소문자/대문자 구분
- 핵심: 실행 권한 있으면 소문자(s,t), 없으면 대문자(S,T)

**quotacheck**
- 문제: 쿼터 파일을 생성하고 점검하는 명령어는?
- 내 답: edquota
- 정답: quotacheck
- 핵심: quotacheck=쿼터 파일 생성/점검, edquota=쿼터 값 편집

**ext2/ext3/ext4**
- 문제: 리눅스에서 저널링을 최초로 지원한 파일시스템은?
- 내 답: 모름
- 정답: ext3
- 핵심: ext2=저널링X, ext3=저널링 최초, ext4=ext3 개선

**XFS**
- 문제: CentOS 7의 기본 파일시스템은?
- 내 답: ext4
- 정답: XFS
- 핵심: XFS=SGI(Silicon Graphics) 개발, CentOS 7 기본

**SMB/CIFS**
- 문제: Windows와 Linux 간 파일 공유에 사용되는 프로토콜은?
- 내 답: NFS
- 정답: SMB/CIFS
- 핵심: NFS(Network File System)=유닉스/리눅스 간, SMB(Server Message Block)=Windows-Linux 간

**/etc/fstab dump 필드**
- 문제: /etc/fstab에서 5번째 필드의 역할은?
- 내 답: fsck 순서
- 정답: dump 백업 여부 (0/1)
- 핵심: 5번째=dump 백업(0/1), 6번째=fsck 순서(0/1/2)

**UUID와 blkid**
- 문제: 디바이스의 UUID와 파일시스템 유형을 확인하는 명령어는?
- 내 답: fdisk -l
- 정답: blkid
- 핵심: blkid=UUID/파일시스템 식별, fdisk -l=파티션 테이블

**저널링**
- 문제: 저널링 파일시스템의 주요 목적은?
- 내 답: 속도 향상
- 정답: 비정상 종료 시 빠른 복구
- 핵심: 저널링은 성능이 아닌 데이터 안정성/복구가 목적

**fstab defaults**
- 문제: /etc/fstab의 defaults 옵션에 포함되는 항목은?
- 내 답: 모름
- 정답: rw,suid,dev,exec,auto,nouser,async
- 핵심: defaults=rw,suid,dev,exec,auto,nouser,async (usrquota 미포함)

**mount/umount**
- 문제: 마운트된 파일시스템을 해제하는 명령어는?
- 내 답: unmount
- 정답: umount
- 핵심: 리눅스에서는 umount (n 하나 빠짐)

**mount -t / mount -o**
- 문제: NFS 파일시스템을 마운트할 때 사용하는 옵션은?
- 내 답: 모름
- 정답: mount -t nfs
- 핵심: -t nfs(NFS 마운트), -o loop(ISO 이미지 마운트)

**fdisk -l**
- 문제: 디스크의 파티션 목록을 출력하는 fdisk 옵션은?
- 내 답: fdisk -a
- 정답: fdisk -l
- 핵심: -l=list(파티션 목록 출력). fdisk -a는 존재하지 않는 옵션.

**e2fsck**
- 문제: ext 계열 파일시스템의 무결성을 검사하는 명령어는?
- 내 답: xfs_repair
- 정답: e2fsck
- 핵심: e2fsck=ext 전용, xfs_repair=XFS 전용

**du 명령어**
- 문제: 디렉토리의 디스크 사용량을 확인하는 명령어는?
- 내 답: df -h
- 정답: du -sh
- 핵심: du=사용량(Disk Usage), df=여유(Disk Free). du -sh=디렉토리 사용량 요약

**setquota vs edquota**
- 문제: 스크립트에서 자동으로 쿼터 값을 설정하는 명령어는?
- 내 답: edquota
- 정답: setquota
- 핵심: edquota=vi 편집기(대화식), setquota=명령행(스크립트 자동화)
